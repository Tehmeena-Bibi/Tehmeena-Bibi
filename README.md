### 🩻 GammaX
**AI-Based X-ray Exposure Quality Assessment**

End-to-end ML pipeline classifying X-rays as under/properly/overexposed using a MobileNetV2 CNN trained via transfer learning on 1000+ images. Combines Logistic Regression for retake validation with Ridge Regression for exposure-adjustment prediction, plus Grad-CAM maps for explainability.

`CNN` `Transfer Learning` `Grad-CAM`

### 🗄️ Venture Management Database
**Enterprise Schema for Ventures & Investments**

Designed a full E-R model to track business ventures, client acquisitions, and investment distributions. Applied multi-level normalization (1NF–3NF) to eliminate redundancy, with complex SQL joins/subqueries in MySQL alongside flexible MongoDB collections.

`SQL` `MySQL` `MongoDB` `Normalization`
